# Inshrah Mumtaz — Portfolio

A personal portfolio website built with React, TypeScript, and Tailwind CSS.

Based on the template by [cupidbity](https://github.com/cupidbity/portfolio-template).

## Tech Stack

- React 19, TypeScript, Vite
- Tailwind CSS, Radix UI, Lucide Icons
- GSAP, React Spring, OGL (WebGL)
- React Router DOM

## Features

- Dark/light mode
- Interactive ASCII text with morphing effects
- Draggable star decorations
- Animated stickers on scroll
- Image carousel for project screenshots
- Responsive design across all devices
